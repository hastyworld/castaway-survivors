# 표류 서바이버 (Castaway Survivors)

무인도에 표류한 생존자가 몰려오는 적을 자동 전투로 쓸어버리고, 살아남아 다음 섬으로 탈출하는 **뱀서라이크(서바이버) 게임**입니다.
기획안(`표류서바이버_기획안.pdf`)의 v1 스코프를 기준으로 만든 **플레이 가능한 초안(v0.1)** 입니다.

- 엔진: **Phaser 3** + **Vite** (100% 코드 → 바이브코딩에 최적)
- 화면: 세로(9:16), 모바일 우선
- 조작: 이동만 (공격은 자동)

---

## 1. 실행 방법 (PC 브라우저에서 바로 확인)

터미널(파워셸)에서 이 폴더로 이동한 뒤:

```powershell
npm install     # 최초 1회 (필요한 라이브러리 설치)
npm run dev      # 개발 서버 실행
```

- 실행되면 터미널에 `http://localhost:5173` 같은 주소가 뜹니다. 브라우저로 열면 게임이 나옵니다.
- **PC 조작**: 방향키 또는 W/A/S/D 로 이동.
- **모바일/터치**: 화면 아무 곳이나 누른 채로 드래그하면 그 자리에 조이스틱이 생깁니다.
- 같은 와이파이의 휴대폰에서도 접속해 실기기 손맛을 볼 수 있습니다(터미널의 `Network:` 주소 사용).

멈추려면 터미널에서 `Ctrl + C`.

## 3. 구글 플레이 출시 준비 (기획안 10단계, 나중에)

웹으로 완성한 뒤 **Capacitor**로 안드로이드 앱으로 포장합니다.

```powershell
npm run build                     # dist/ 에 웹 결과물 생성
npm install @capacitor/core @capacitor/cli @capacitor/android
npx cap init                      # capacitor.config.ts 이미 있음
npx cap add android
npx cap sync
npx cap open android              # Android Studio에서 열려 → APK/AAB 빌드
```

이후 필요한 것:
- 구글 플레이 개발자 계정($25, 1회)
- 앱 아이콘 / 스플래시 / 스토어 스크린샷
- 보상형 광고는 AdMob 플러그인으로 **맨 마지막에** 연동 (기획안 7·8번: 필수 광고 0개)
